Home Assistant
====================================

.. image:: media/home_asstis2.jpg
    :align: center

Home Assistant是一个基于Python的开源智能家居系统，支持多个品牌的智能家居设备，可以轻松实现设备的语音控制和自动化。您可以在树莓派上运行Home Assistant，而无需将数据发送到互联网上。这意味着您可以完全控制自己的数据，限制来自智能设备的互联网流量，并加强安全性。


RasPad 3 由树莓派4供电，配备 10.1 英寸高清触摸屏和 5 小时电池续航时间。我们还为其编写了自启动和全屏启动脚本，因此您可以更轻松、更快速地在 RasPad 上使用Home Assistant。

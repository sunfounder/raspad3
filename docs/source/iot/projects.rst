项目
=============

在本节中，您将开始学习创建 IoT 项目。

在开始项目之前，应该确保已经下载了我们提供的代码。

打开终端并输入以下命令从 github 上下载。

.. raw:: html

    <run></run>

.. code-block::

    git clone https://github.com/sunfounder/raspad3.git

然后将代码从 ``iot`` 文件夹复制到 ``cloud4rpi-raspberrypi-python`` 文件夹。

.. raw:: html

    <run></run>

.. code-block:: shell

    cp -r /home/pi/raspad3/iot/* /home/pi/cloud4rpi-raspberrypi-python
